## Reels script (~30–45 sec)

**[Hook — 0–3s]**  
“Founders: if your company memory is scattered across apps, your AI can’t help you think—only react.”

**[Problem — 3–12s]**  
“You need **Markdown in Obsidian**, structured capture, and a **local agent** that reads *your* vault—not generic web answers.”

**[Solution — 12–28s]**  
“We’re running a **7-day Seattle cohort**: **Obsidian** + **self-hosted agent** (e.g. OpenClaw-style) + **DIKW**. Security built in. Educational only.”

**[CTA — 28–45s]**  
“Link in bio · `{{signup_url}}` · Not affiliated with Anthropic/Obsidian/YouTube.”

**On-screen text:** `Seattle · Obsidian · Local AI · 7 days`
